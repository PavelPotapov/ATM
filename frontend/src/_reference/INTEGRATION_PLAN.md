# План интеграции data-table-filters

## Этап 1: Изучение структуры

### Компоненты для изучения:

1. **DataTable** - основной компонент таблицы
2. **Filters** - компоненты фильтрации
3. **Search** - глобальный поиск
4. **ColumnVisibility** - управление видимостью колонок
5. **RowDetails** - сайдбар с деталями строки
6. **Pagination** - пагинация
7. **LiveMode** - режим реального времени

## Этап 2: Адаптация под FSD

### Структура в нашем проекте:

```
widgets/
  table/
    ui/
      DataTable/          # Базовый компонент (уже есть)
      Filters/             # Фильтры (новое)
      Search/              # Глобальный поиск (новое)
      ColumnVisibility/    # Управление колонками (новое)
      RowDetails/          # Сайдбар с деталями (новое)
      Pagination/          # Пагинация (новое)
    lib/
      useTableFilters.ts   # Хуки для фильтрации
      useTableSearch.ts    # Хуки для поиска
      useTableSort.ts      # Хуки для сортировки
```

## Этап 3: Интеграция с нашим API

### Адаптация под EstimateTable:

1. Использовать наши хуки (`useEstimateTableData`, `useUpdateCell`)

## Этап 4: Дополнительные функции

1. **Live режим** - для будущей интеграции с WebSockets
2. **Совместное редактирование** - когда несколько пользователей работают одновременно
