# Референсный код для изучения

## data-table-filters

Здесь будет размещен код из репозитория [openstatusHQ/data-table-filters](https://github.com/openstatusHQ/data-table-filters) для изучения и адаптации.

### Как скопировать код:

```bash
# Из корня проекта (C:\projects\ATM)
cd frontend/src/_reference
git clone https://github.com/openstatusHQ/data-table-filters.git
```

Или просто скопируйте нужные файлы из репозитория в эту папку.

### Что нужно изучить:

1. **Структура компонентов** - как организованы компоненты таблицы
2. **Фильтры** - реализация фильтрации по столбцам
3. **Глобальный поиск** - поиск по всем данным
4. **Сортировка** - сортировка по столбцам
5. **Пагинация** - пагинация данных
6. **Выделение строк** - выбор строк
7. **Сайдбар** - боковая панель с деталями строки
8. **Контроль колонок** - показ/скрытие колонок
9. **Live режим** - обновление данных в реальном времени (для будущих сокетов)

### План интеграции:

1. Изучить структуру кода
2. Выделить переиспользуемые компоненты
3. Адаптировать под наш проект (FSD архитектура)
4. Интегрировать с нашим API
5. Добавить функциональность редактирования ячеек
6. Добавить проверку прав доступа
