# Постоянный ngrok URL

## Важно

**С текущим решением (относительные пути) постоянный URL не обязателен!**

Фронтенд автоматически использует текущий домен, поэтому ngrok URL может меняться - ничего обновлять не нужно.

---

## Если нужна постоянная ссылка

Если вам **обязательно нужна постоянная ссылка** (например, для демонстрации клиенту):

### Шаг 1: Перейдите на платный план

1. Зарегистрируйтесь на https://ngrok.com
2. Перейдите в раздел Pricing
3. Выберите план (например, "Personal" - $8/месяц)
4. Оплатите подписку

### Шаг 2: Зарезервируйте домен

1. Перейдите в Dashboard → Reserved Domains
2. Нажмите "Add Domain"
3. Введите желаемый поддомен (например, `myapp`)
4. Получите домен вида `myapp.ngrok-free.app` или `myapp.ngrok.app`

### Шаг 3: Используйте зарезервированный домен

```bash
# Для ngrok-free.app
ngrok http 3000 --domain=myapp.ngrok-free.app

# Или для ngrok.app (платный)
ngrok http 3000 --domain=myapp.ngrok.app
```

**Результат:** URL всегда будет `https://myapp.ngrok-free.app` (или `https://myapp.ngrok.app`)

---

## Итог

**Для большинства случаев:** Используйте обычный ngrok - всё работает автоматически с любым URL.

**Если нужна постоянная ссылка:** Используйте платный ngrok с зарезервированным доменом ($8/месяц).
